# 🚀 Goal Tracker

A modern full-stack **Employee Goal Management & Performance Tracking Portal** built for hackathons, internship projects, and enterprise workflow automation.

Goal Tracker helps organizations streamline:
- Goal Creation
- Goal Approval Workflow
- Quarterly Check-ins
- Employee Performance Tracking
- Analytics & Reporting

# 👥 User Roles

## 👨‍💼 Employee
- Create goals
- Edit goals
- Submit goals for approval
- Add quarterly updates
- Track progress

## 👨‍💻 Manager
- Review employee goals
- Approve / reject goals
- Monitor team progress
- Add quarterly review comments

## 🛠 Admin
- View analytics dashboard
- Monitor organization performance
- Export reports
- Access system-wide statistics
